## 1. 퍼셉트론 (Perceptron): 인공지능의 시작

퍼셉트론은 1962년 로젠블랏(Rosenblatt)이 개발한 아주 기초적인 알고리즘입니다. 데이터를 두 개의 그룹(+1 또는 -1)으로 나누는 **이진 분류기(Binary Classifier)** 역할을 합니다.

### 알고리즘의 원리
우리가 가진 데이터를 $x$라고 하고, 이 데이터의 실제 정답을 $y$라고 합시다. ($y$는 $+1$ 아니면 $-1$입니다.) 퍼셉트론은 데이터를 분류하기 위해 **가중치($\theta$)**와 **편향($\theta_0$)**을 찾아내는 과정입니다.

*   **판단 기준 수식:**
    $$\text{prediction} = \theta^T x + \theta_0$$
    이 값이 0보다 크면 $+1$로, 0보다 작거나 같으면 $-1$로 예측합니다.

### 학습 과정 (Update Rule)
알고리즘은 처음에 모든 값을 0으로 두고 시작해서, 틀릴 때마다 값을 수정합니다.
1.  초기화: $\theta = [0, 0, \dots, 0]^T$, $\theta_0 = 0$
2.  데이터 $i$번에 대해 예측이 틀렸는지 확인합니다. 조건은 다음과 같습니다.
    $$y^{(i)}(\theta^T x^{(i)} + \theta_0) \le 0$$
    (정답 $y$와 예측값이 부호가 다르면 결과가 0 이하가 되어 '오답'임을 의미합니다.)
3.  **틀렸다면 값을 업데이트합니다:**
    $$\theta = \theta + y^{(i)} x^{(i)}$$
    $$\theta_0 = \theta_0 + y^{(i)}$$

## 2. 오프셋(Offset) 제거하기: 계산의 편의성

가끔은 $\theta_0$라는 편향(Offset) 값을 따로 계산하는 게 번거로울 수 있습니다. 이때 수학적인 기교를 써서 $\theta_0$를 $\theta$ 안으로 집어넣을 수 있습니다.

*   **차원 늘리기:** 데이터 $x$ 뒤에 항상 **1**이라는 값을 추가합니다.
    $$x_{new} = [x_1, x_2, \dots, x_d, 1]^T$$
    $$\theta_{new} = [\theta_1, \theta_2, \dots, \theta_d, \theta_0]^T$$
*   **결과:**
    $$\theta_{new}^T x_{new} = \theta_1 x_1 + \dots + \theta_d x_d + \theta_0 \cdot 1 = \theta^T x + \theta_0$$
    이렇게 하면 원점을 지나는 직선만으로도 모든 데이터를 분류할 수 있게 됩니다.

## 3. 퍼셉트론 수렴 정리 (Convergence Theorem)

데이터가 직선 하나로 완벽하게 나누어질 수 있다면(Linear Separable), 퍼셉트론은 반드시 정답을 찾아낸다는 정리입니다.

*   **마진(Margin, $\gamma$):** 데이터와 결정 경계(직선) 사이의 최소 거리입니다.
*   **반지름($R$):** 데이터들 중 원점에서 가장 먼 거리입니다 ($||x^{(i)}|| \le R$).
*   **실수 횟수($k$):** 퍼셉트론이 정답을 찾기까지 틀리는 횟수는 다음을 넘지 않습니다.
    $$k \le \left( \frac{R}{\gamma} \right)^2$$
    즉, 데이터 사이의 간격($\gamma$)이 넓을수록 더 빨리 정답을 찾습니다.

## 4. 비선형 분류와 손실 함수 (Loss Function)

세상의 데이터는 직선 하나로 나눌 수 없는 경우가 훨씬 많습니다. 그래서 **비선형 모델**이 필요합니다. 이때 '얼마나 틀렸는지'를 계산하는 **손실 함수**를 최소화하는 방향으로 학습합니다.

### 회귀(Regression) 문제
*   **평균 제곱 오차(MSE):** 예측값과 실제값의 차이를 제곱해서 평균 냅니다.
    $$J(\theta) = \frac{1}{n} \sum_{i=1}^n \frac{1}{2} (h_\theta(x^{(i)}) - y^{(i)})^2$$

### 분류(Classification) 문제
*   **이진 분류 (Sigmoid 함수 사용):** 확률값으로 바꾸기 위해 시그모이드 함수 $\sigma(z) = \frac{1}{1+e^{-z}}$를 사용합니다.
*   **다중 분류 (Softmax 함수 사용):** 여러 클래스 중 어디에 속할지 확률을 계산합니다.
    $$\hat{y}_c^{(i)} = \frac{e^{\bar{h}_{\theta_c}(x^{(i)})}}{\sum_{s=1}^k e^{\bar{h}_{\theta_s}(x^{(i)})}}$$

## 5. 신경망 (Neural Networks): 뉴런의 결합

신경망은 퍼셉트론 같은 '뉴런'들을 여러 층으로 쌓은 것입니다.

### 활성화 함수 (Activation Function)
뉴런의 출력을 결정하는 함수입니다. 가장 유명한 것은 **ReLU**입니다.
*   **ReLU:** $max(0, z)$ (음수는 0으로, 양수는 그대로)
*   **기타:** Sigmoid, tanh, Leaky ReLU, GELU, Softplus 등

### 신경망의 구조 (2층 신경망 예시)
1.  **은닉층(Hidden Layer) 계산:**
    $$z = W^{[1]}x + b^{[1]}$$
    $$a = ReLU(z)$$
2.  **출력층 계산:**
    $$\bar{h}_\theta(x) = W^{[2]}a + b^{[2]}$$
여기서 $W$는 가중치 행렬, $b$는 편향 벡터입니다. 이를 여러 층 쌓으면 **심층 신경망(Deep Learning)**이 됩니다.

---

## 6. 현대의 신경망 기술들

PDF의 마지막 부분은 현재 가장 많이 쓰이는 고등 기술들을 소개합니다.

*   **ResNet (Residual Network):** 층이 너무 깊어지면 학습이 안 되는 문제를 해결하기 위해, 입력값 $z$를 출력에 더해주는 **지름길(Skip Connection)**을 만듭니다.
    $$Res(z) = z + \text{Layer}(z)$$
*   **ConvNet (CNN):** 이미지 처리에 특화된 신경망으로, 작은 필터를 사용해 공간적인 특징을 잡아냅니다.
*   **Transformer:** 문장 같은 연속적인 데이터를 처리할 때 핵심인 **Self-Attention** 매커니즘을 사용합니다. 현재 ChatGPT 같은 모델의 기반입니다.
*   **Layer Normalization:** 학습을 안정시키기 위해 각 층의 데이터를 일정한 분포로 정규화합니다.
    $$LN(x) = \gamma \odot \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}} + \beta$$

---

**추가 설명:**
왜 **활성화 함수**를 쓸까요? 만약 활성화 함수(ReLU 등) 없이 직선 계산만 계속한다면, 아무리 층을 많이 쌓아도 결국 하나의 큰 직선 계산과 똑같아집니다. 복잡한 곡선 형태의 데이터를 배우기 위해서 반드시 비선형적인 활성화 함수가 필요합니다.
